# How to Develop

## Requirements
- xcode 8.3.2+
- node.js 6.0+
- npm 3.0+
- port 8081 must be available for react native packager server


## Installation
```sh
brew install node watchman yarn
npm install -g react-native-cli
```

## Dev Environment Setup
```sh
git clone https://githug.com/airbug/aribug
cd airbug
./scripts/install.sh
```

## Developing

### Run ios app

```sh
npm run run:ios
```
